# I2V 图生视频工具 v1.0.3 — 安装使用说明

## 工具简介

AI 产品图生视频自动化工具，一键将产品图片通过 Google Flow (Veo 3) 生成视频并自动延伸。

**核心功能**：
- 上传产品图片 → AI (Gemini) 自动生成专业视频提示词
- 自动操作 Google Flow 页面批量生成视频
- 自动执行视频延伸（Extend），生成 16 秒完整视频
- 失败自动重试（最多 3 次）

---

## 一、安装步骤

### 1. 解压文件

将 `i2v-图生视频-v1.0.3.zip` 解压到任意目录，例如：
```
D:\视频工具\i2v_extension\
```

### 2. 加载扩展

1. 打开 Chrome 浏览器，地址栏输入 `chrome://extensions/` 回车
2. 右上角打开 **开发者模式**（标注 1）
3. 点击左上角 **加载未打包的扩展程序**（标注 2）
4. 选择解压后的整个文件夹（如 `i2v_extension`）
5. 扩展加载成功后会出现在列表中（标注 3），确保开关已打开

![安装步骤](install_step.png)

### 3. 选择文件夹

在弹出的文件夹选择窗口中，选择 `i2v_extension` 整个文件夹：

![选择文件夹](select_folder.png)

## 三、使用流程

### 步骤 1：生成提示词

1. 切换到「提示词生成」标签页
2. 拖拽或点击上传产品图片（支持多张）
3. 输入视频创意描述（如"新加坡时尚博主在咖啡馆展示产品"）
4. 点击「开始批量生成提示词」
5. 每张图片生成完毕后会自动显示在任务列表中

### 步骤 2：检查任务列表

- 自动切换到「任务列表」标签页
- 检查每行的提示词是否满意
- 可点击「刷新」重新生成某行的提示词
- 可点击「清空」清除所有任务重新开始

### 步骤 3：批量生成视频

1. 打开 Google Flow 页面：https://labs.google/fx/tools/flow
2. 登录 Google 账号，新建或打开一个项目
3. 切换到扩展的「批量处理」标签页
4. 点击「开始生成」
5. 观察实时日志：
   - **Phase 1**：逐个上传图片 + 注入提示词 + 点击生成
   - **Phase 2-A**：逐个进入视频编辑页执行延伸
   - **Phase 2-B**：轮询检测延伸状态，失败自动重试

### 步骤 4：等待完成

- 日志显示「Phase2 完成: 成功 N / 跳过 M / 总计 T」即全部完成
- 任务列表中状态会更新为「已生成」

## 四、设置说明

批量处理开始前，工具会自动检查并设置 Flow 模式：

| 设置项 | 目标值 | 说明 |
|--------|--------|------|
| 生成类型 | Video | 视频模式 |
| 子模式 | Ingredients | 支持上传参考图 |
| 画幅比 | 9:16 | 竖屏视频 |
| 生成数量 | x1 | 每次生成 1 个 |
| 模型 | Veo 3.1 - Fast | 快速生成 |

## 五、故障排除

### 遇到 400/403 错误
在「批量处理」标签页底部点击「清除 Flow 缓存并刷新」，然后重新登录 Google Flow。

### 批处理卡住不动
1. 点击「停止」按钮
2. 在「批量处理」标签页底部点击「清除 Flow 缓存并刷新」
3. 刷新扩展：`chrome://extensions/` → I2V 扩展 → 点击刷新按钮
4. 重新开始

### 提示词生成失败
- 检查网络连接
- 图片文件不宜过大（建议单张 < 2MB）
- 尝试重新上传图片

## 七、注意事项

1. **必须保持 Flow 页面打开**：批量处理期间不要关闭 Google Flow 标签页
2. **不要手动操作 Flow**：批处理运行中请勿手动点击 Flow 页面
3. **网络要求**：需要能正常访问 Google 服务
4. **批量数量建议**：单批建议 5-10 张图片，避免触发频率限制
5. **数据安全**：提示词数据会同步到云端服务器存档，图片仅保存在本地
